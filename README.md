# Claude Code ハンズオン

Claude Code の基本機能を体験するためのハンズオン教材です。  
実際のファイル操作・データ分析・ルール設定・スキル活用・API連携・Web システム作成まで、段階的に学べる構成になっています。

## ハンズオン内容

### 01. ファイル整理

散らばったファイル（画像・テキスト・CSV など）を、種類ごとにフォルダ分け・リネームして整理する演習です。

- 対象: [01_ファイル整理/](01_ファイル整理/)
- プロンプト: [プロンプト集/01_ファイル整理/整理実施.md](プロンプト集/01_ファイル整理/整理実施.md)

### 02. CSV 集計

売上データ CSV を分析し、月別売上合計やカテゴリ別ランキングを算出する演習です。

- 対象: [02_CSV集計/sales_data.csv](02_CSV集計/sales_data.csv)
- プロンプト: [プロンプト集/02_CSV集計/売上集計.md](プロンプト集/02_CSV集計/売上集計.md)

### 03. ルール

Claude Code のルール機能（`.claude/rules/`）を活用する演習です。参加者リストのグループ化や、関西弁での応答ルール作成を体験します。

- 対象: [03_ルール/参加者リスト.txt](03_ルール/参加者リスト.txt)
- プロンプト:
  - [参加者グループ化](プロンプト集/03_ルール/参加者グループ化.md)
  - [関西弁ルール作成](プロンプト集/03_ルール/関西弁ルール作成.md)

### 04. スキル

Claude Code のスキル機能（`.claude/skills/`）を活用する演習です。打ち合わせメモから議事録を自動生成するスキルなどを体験します。

- 対象: [04_スキル/](04_スキル/)
- スキル定義: [プロンプト集/ルール・スキル・エージェント/skills/](プロンプト集/ルール・スキル・エージェント/skills/)

### EX01. Web システム作成（応用）

売上データ CSV からブラウザで閲覧できるダッシュボード（HTML）を生成する応用演習です。

- プロンプト: [売上ダッシュボード](プロンプト集/EX01_Webシステム/売上ダッシュボード.md)

### 06. API 連携

外部 API（郵便番号検索）を呼び出して結果を取得する演習です。

- プロンプト: [郵便番号検索](プロンプト集/06_API連携/郵便番号検索.md)

## ディレクトリ構成

```
├── 01_ファイル整理/       # 演習1: 整理対象のファイル群
├── 02_CSV集計/            # 演習2: 売上データ CSV
├── 03_ルール/             # 演習3: 参加者リスト
├── 04_スキル/             # 演習4: 打ち合わせメモ
├── プロンプト集/           # 各演習のプロンプト・スキル定義
├── output/                # 各演習の出力先（プロンプト実行時に作成される）
├── 参考_作成物/            # 作成物のサンプル
├── 準備用/                # ダミーデータ作成用スクリプト
└── .claude/               # Claude Code 設定（ルール・スキル）
```

## 前提条件

- [Claude Code](https://docs.anthropic.com/en/docs/claude-code) がインストール済みであること
